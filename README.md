# QSC UI

## Usage

To use, load this script in any location in your page.

To config, add the following code before lodding this script:

```html
<script>
qscUIConfig = {
    autoAppendNav: true,
    autoAppendStatistics: true
}
</script>
```

## Default Opinions

```javascript
var opinions = {
    autoAppendNav: true,
    autoAppendStatistics: true
}
```
