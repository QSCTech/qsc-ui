# QSC UI

## Usage

To use, load this script in any location in your page.

```html
<script src="http://static.myqsc.com/js/qsc-ui.js"></script>
```

To config, add the following code before lodding this script:

```html
<script>
qscUIConfig = {
    autoAppendNav: true,
    autoAppendStatistics: true
}
</script>
```

## Default Opinions

```javascript
var opinions = {
    autoAppendNav: true,
    autoAppendStatistics: true
}
```
